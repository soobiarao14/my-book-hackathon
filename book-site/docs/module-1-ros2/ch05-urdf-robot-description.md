# Chapter 5: URDF Robot Description

:::info Status
This chapter is currently under development.
:::

## Overview

This chapter teaches you how to describe robots using URDF (Unified Robot Description Format).

## Learning Objectives

By the end of this chapter, you will:
- Understand URDF structure and syntax
- Create robot models with links and joints
- Add visual and collision geometries
- Generate URDF with Xacro macros

## Topics Covered

1. URDF Fundamentals
2. Links and Joints
3. Visual and Collision Models
4. Xacro for Robot Descriptions
5. Robot State Publisher
